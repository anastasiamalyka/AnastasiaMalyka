  ## Запитання для самоконтролю
1. Як встановити середовище розробки Python? Як запустити виконання програми?
Треба завантажити останюю версію Python з офіційного сайту, в інсталяторі обов’язково поставити галочку "Add Python to PATH". Також треба встановити середовище розробки (Visual Studio Code, PyCharm, тп).
Щоб запустити виконання програми треба створити новий файл в середовищі розробки, написати програму та натиснути Run (або F5).
2. Які основні відмінності в роботі під ОС Windows та Unix-подібнними системами?
Windows орієнтована на графічний інтерфейс і простоту для користувача, Unix-подібні системи — на командний рядок і гнучке налаштування. Відрізняються команди, структура файлів і спосіб встановлення програм.
3. Поясніть структуру найпростішої програми на Python 3
Найпростішою програмою є =print("Привіт!")=. Вона складається з команди *print*, що виводить результат на екран, та самого тексту, який виводиться на екран.
4. Як здійснюються базові операції вводу-виводу в Python 3? Які ще часто викорстовувані операції (і відповідні оператори Python 3) Вам відомі?
Операція вводу від користувача здійснюється командою *input()*, а виводу командою *print()*. Часто використовувані операції: Арифметичні: + (додавання), - (віднімання), * (множення), / (ділення), ** (степінь). Порівняння:  <, >, <=, >=.
5. Які Ви знаєте команди Git?
Команди, які мені відомі: git status – подивитися стан файлів у репозиторії, git commit -m "повідомлення" – зафіксувати зміни, git push – відправити зміни на віддалений репозиторій, git init – створити новий репозиторій.
